Installation/Updates
####################


.. _installation-pip:

Using PIP
*********

Installation from PyPI using PIP
================================

.. code-block:: bash

   pip3 install pyEDAA.Launcher


Updating from PyPI using PIP
============================

.. code-block:: bash

   pip3 install -U pyEDAA.Launcher


Uninstallation using PIP
========================

.. code-block:: bash

   pip3 uninstall pyEDAA.Launcher


Installation from local directory using PIP
===========================================

.. code-block:: bash

   pip3 install .


.. _installation-setup:

Using ``setup.py`` (legacy)
***************************

Installation using ``setup.py``
===============================

.. code-block:: bash

   setup.py install
